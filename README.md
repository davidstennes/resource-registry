# resource-registry

Private, invite-only registry.

This repository uses GitHub Discussions as the registry.
Members add and manage their own information.

## How it works
- You are invited to this repository.
- Use Discussions to add or update:
  - Your skills / background
  - Offers (items, tools, time)
  - Needs (help, items, requests)

## Guidelines
- Pseudonyms encouraged.
- No exact locations or real-time tracking.
- No phone numbers or personal contact info.
- Edit or delete your own posts as needed.
